# Big O & Objects
Objects are useful for storage of data where order doesn't matter and you need fast insertion and removal.
- In JS objects function as a hashmap

## Runtimes
- insertion &rarr; O (1)
- removal &rarr; O (1)
- search &rarr; O (n)
- access &rarr; O (1)
